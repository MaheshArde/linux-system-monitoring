# Linux System Monitoring

A beginner-friendly Linux System Administration project to practice monitoring system resources, processes, services, and logs on Ubuntu Server.

## Project Objectives

- Monitor CPU usage
- Monitor memory usage
- Check disk usage
- Monitor running processes
- Check service status
- View system logs
- Create a basic Bash monitoring script

## Technologies Used

- Ubuntu Server 24.04 LTS
- Bash
- Linux Monitoring Commands
- Git & GitHub
